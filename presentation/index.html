¿Cómo se Determina la Criticidad?

Sistema de clasificación basado en CVSS y análisis contextual

📊 CVSS Score (Common Vulnerability Scoring System)

Es la métrica base que evalúa la vulnerabilidad de forma técnica:

9.0 - 10.0
🔴 CRÍTICO
Explotable remotamente, sin autenticación, impacto máximo
7.0 - 8.9
🟠 ALTO
Impacto significativo, fácil de explotar, requiere poco conocimiento
4.0 - 6.9
🟡 MEDIO
Impacto moderado, requiere ciertos requisitos previos
0.1 - 3.9
🟢 BAJO
Impacto mínimo, difícil de explotar, o requiere permisos

🔍 Factores de CVSS Considerados

Vector de Acceso

Remoto: Máximo riesgo
Local: Riesgo menor
Física: Riesgo mínimo

Complejidad de Ataque

Baja: Se explota fácilmente
Alta: Requiere bypass de seguridad

Privilegios Requeridos

Ninguno: Máximo riesgo
Usuario: Riesgo medio
Admin: Riesgo bajo

Impacto de Confidencialidad

Alto: Datos sensibles expuestos
Parcial: Algunos datos afectados
Ninguno: Sin exposición

Impacto de Integridad

Alto: Modificación sin restricción
Parcial: Cambios limitados
Ninguno: No afectada

Impacto de Disponibilidad

Alto: Servicio no disponible
Parcial: Degradación de servicio
Ninguno: No afectada

🏦 SQL Injection en Transferencias

Función: transferMoneyBetweenAccounts() - Transferencias bancarias

Detectado en: banking/transfers.js:123

❌ Código Vulnerable: 
const query = `SELECT balance FROM accounts WHERE id = ${accountId} AND pin = '${userPin}'`;
const result = db.query(query);
// Atacante: accountId = "1 OR 1=1" para sortear validación
✅ Solución: 
const query = 'SELECT balance FROM accounts WHERE id = ? AND pin = ?';
const result = db.query(query, [accountId, userPin]);
// Prepared statement - impide inyección

Impacto: 💰 Acceso a todas las cuentas bancarias sin validación

Severidad: CRÍTICO

CVSS Score: 9.8

02

🏦 XSS en Dashboard de Trasferencias

Función: renderTransferHistory() - Mostrar historial de transferencias

Detectado en: banking/dashboard.jsx:156

❌ Código Vulnerable: 
const TransactionNote = ({note}) => (
  <div dangerouslySetInnerHTML={{__html: note}} />
);
✅ Solución: 
const TransactionNote = ({note}) => (
